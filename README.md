# Wedding Invitation Web App (Template)

Mobile-first, scrollable wedding invitation built with React + Vite, Tailwind CSS, and Framer Motion.

## Run locally

- Install: `npm install`
- Dev server: `npm run dev`
- Production build: `npm run build`

## Customize content

- Edit all copy and data in `src/inviteData.json`.
- (Optional) Drop your hero background image at `public/backgrounds/hero.jpg` (or update the path in `src/inviteData.json`).
- `src/weddingData.js` maps the JSON into the view-model used by components.
- Sections live in `src/components/`.
- Placeholder illustrations are in `src/assets/illustrations/`.
